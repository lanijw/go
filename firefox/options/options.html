Due to Firefox bug 1378527 it's not possible to use a file upload dialog in the popup window of a Firefox Add-On. As soon as the user interacts with the file upload dialog, the Add-On popup gets closed, disconnecting the listeners on the input field. Once the user selects a file, nothing is notified of that change and the result if a lack of response from the system. As an alternative you can upload your configuration through this options page.
